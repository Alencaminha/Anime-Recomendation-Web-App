SOBRE O PROJETO

Este trabalho teve como objetivo criar uma aplicação com um CRUD e utilizar uma API Rest externa. Para isto foi utilizado o Node.js com o Express para criar um backend com APIs que façam conexão com o banco de dados escrito em SQLite, além da API Jikan, a maior Rest API de animes do mundo, que conta com todo o conteúdo do site My Anime List. A aplicação permite que o usuário insira um anime que já assistiu e recebe uma recomendação de um anime relacionado, assim como algumas informações e imagem sobre o mesmo.
