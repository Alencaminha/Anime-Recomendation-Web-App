SOBRE

Este trabalho tem como objetivo criar uma aplicação com um CRUD e utilizar uma API Rest externa. Aplicação de usuabilidade para Website e Mobile, tornando a experiência do usuário mais simples, intuitiva e agil ao navegar pelas cinco telas proposta neste trabalho. Para isto foi utilizado o Node.js com o Express para criar um backend com APIs que façam conexão com o banco de dados escrito em SQLite, além da API Jika - Myanimelist.
A aplicação permite que o usuário insira um anime que já assistiu e recebe uma recomendação de um anime relacionado, assim como algumas informações e imagem sobre o mesmo.
