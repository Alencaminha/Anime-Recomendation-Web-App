SOBRE

Este trabalho tem como objetivo realizar a aplicação de CRUD (Create, Read, Update e Delete) que é um acrônimo para as maneiras de se operar informação armazenada em banco de dados.
O site propõe recomendar aos usuários um anime para assistir com base nos cinco animes favoritos sinalizados pelo usuário na hora da consulta.
Foi utilizado na elaboração desta A3 o MYSQL e API Jikan - Myanimelist.
