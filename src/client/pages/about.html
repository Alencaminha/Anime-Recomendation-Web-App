SOBRE

Este trabalho tem como objetivo realizar a aplicação de CRUD (Create, Read, Update e Delete) que é um acrônimo para as maneiras de se operar informação armazenada em banco de dados. Além da aplicação de usuabilidade para Website e Mobile, tornando a experiência do usuário mais simples, intuitiva e agil ao navegar pelas cinco telas proposta neste trabalho. Utilizando na elaborado no projeto o MySQL, juntamente como a API Jika - Myanimelist.
O site propõe recomendação de anime ao usuário assistir, com base animes favorito sinalizado pelo usuário na hora da consulta.
